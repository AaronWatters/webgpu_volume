Example HTML pages.

The following pages exercise the webgpu_volume functionality for demonstration and debugging.

Some pages generate visual outputs, and others only send output to the Javascript console.

try_threshold_view -- threshold projection using view abstraction.

try_mix_view -- shaded segmentation using view abstraction.

try_view -- max value projection using view abstraction.

try_pipeline -- max value (historical).

try_threshold -- isosurface.

try_mix -- indexed colorization mixed with isosurface shading.
